# Local scratch directory

This directory is **gitignored** (except `.gitkeep` and this README).
Use it for working files that:

- need to survive a reboot (unlike `/tmp`),
- relate to this repository (drafts of issue comments, release-PR bodies,
  exporter `/metrics` captures, generated logs, etc.),
- should **never** end up on GitHub.

Anything you put in here stays on your machine.
